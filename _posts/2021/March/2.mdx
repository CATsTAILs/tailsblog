# UMAP

Give __fuzzy topological structure__ in the high dimensional point cloud.

find the __fuzzy topological structure__ to a point cloud in a low dimensional euclidean space which minimize difference between it and the given metrics.

the objective function is formed as follows:
$$
\sum_{e \in E} w_h(e) \log(\frac{w_h(e)}{w_l(e)})) + (1-w_h(e) \log(\frac{1-w_h(e)}{1-w_l(e)})
$$
where $w_h(e)$ is a weigth of an edge $e$ in high-dim space, __i.e.__ given point cloud. And $w_l(e)$ is a weight of an edge $e$ in low-dim space.

Effective KNN calculation algorithm, differentiable apporximation of an objective function, etc. are used to improve a performance of __UMAP__.
